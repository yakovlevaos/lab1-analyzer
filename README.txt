ПРОЕКТ ПОДГОТОВЛЕН В РАМКАХ ЛАБОРАТОРНОЙ РАБОТЫ
=================================================

Тема: Синтаксический анализ описаний переменных (Pascal)
Используемые технологии: Next.js, React, Tailwind CSS, JavaScript (RegExp)

-------------------------------------------------
ИНСТРУКЦИЯ ПО ЗАПУСКУ ЛОКАЛЬНО
-------------------------------------------------

1. Установите Node.js
   Убедитесь, что установлена версия Node.js не ниже 18.
   Проверка версии:
   node -v
   npm -v

   Если не установлено — скачайте с официального сайта:
   https://nodejs.org/

2. Распакуйте архив проекта
   Распакуйте lab1-analyzer.zip в удобную директорию, например:
   C:\Projects\lab1-analyzer

3. Установите зависимости
   Откройте терминал в папке проекта и выполните команды:
   cd lab1-analyzer
   npm install

   После этого автоматически установятся все необходимые пакеты.

4. Запустите проект
   Введите команду:
   npm run dev

   После сборки приложение станет доступно по адресу:
   http://localhost:3000

5. Проверьте работу
   На странице откроется интерфейс для проверки описаний переменных.
   Вставьте пример кода на Pascal и нажмите «Проверить».

6. Остановка проекта
   Чтобы завершить выполнение, нажмите Ctrl + C в терминале.

-------------------------------------------------
Минимальные требования:
- Node.js v18+
- npm (устанавливается вместе с Node.js)
- Современный браузер (Chrome, Firefox, Edge и т.д.)
-------------------------------------------------
